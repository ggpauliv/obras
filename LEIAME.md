# Pawliv Obras — Protótipo Navegável

## Como rodar localmente

### Opção 1 — Clique duplo
Abra o arquivo `INICIAR.bat` (clique duplo)

### Opção 2 — Terminal
```
npm start
```

O sistema abrirá em: http://localhost:3000

## Navegação

O sistema abre na tela de Login. Clique em **Entrar** para acessar o Dashboard.

Use o **botão ☰** no canto superior direito para navegar entre todas as telas.
